# Major Assignment 1 – GitLab and Bitwise Operators 10/17/25(new repository)
## Description 

This project is a C program that implements a menu driven interface for performing bitwise operations on 32-bit unsigned integers (1 to 4294967295). The operations include counting leading zeros, endian swapping, right bit rotation, and parity computation, each implemented using only C bitwise operators without relying on built-in or library functions. The project is structured across six files: a header file (major1.h) defining function prototypes, a main file (major1.c) handling the menu system and user input, and four operation-specific files (clz.c, endian.c, rotate.c, parity.c), each managed by a team member.
